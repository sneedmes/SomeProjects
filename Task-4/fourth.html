Задача 4.. Создать функцию, которая группирует студентов по возрасту.

Требуется написать функцию, которая отвечает следующим требованиям: Функция принимает 2 аргумента - массив из уникальных целых чисел и сумму в виде целого числа. Если сумма двух любых чисел массива из 1-го аргумента равна числу, которое приходит 2-м аргументом, функция должна вернуть новый массив из этих двух чисел в любом порядке. Если решения нет, вернуть пустой массив.
