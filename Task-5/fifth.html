Задача 5. Получить единый массив из любимых пицц каждого друга

На старте, указываем, что начальное значение переменной accum - пустой массив []. Далее, последовательно "бежим" по каждому элементу массива и каждый раз возвращаем обновленное значение accum. Обновленное значение accum это - массив, в который при каждой итерации добавляются: Все текущие элементы переменной accum, которые были добавлены в предыдущей итерации Все пиццы из текущего элемента массива в переменной current: current.pizzas
